# Flowdesk Task & Project Management API

Flowdesk is a simple **Task and Project Management System** API built with **.NET** (backend) and **React** (frontend). This API allows users to manage projects, tasks, and their workflows efficiently.

## Features

- Create, update, view, and delete **projects**  
- Create, update, view, and delete **tasks** under projects  
- Assign tasks and track progress  
- Validation for required fields in both projects and tasks  
- Confirmation before deleting records (via modal in frontend)  

---

## Technology Stack

- **Backend:** .NET 6+ Web API  
- **Frontend:** React, TypeScript, React-Bootstrap  
- **Database:** SQL Server / PostgreSQL (configurable in `appsettings.json`)  
- **Authentication:** JWT-based authentication  
- **Testing:** xUnit / Postman for API testing
